*# Despliegue Automatizado EVE-NG*



*Proyecto de Prácticas Preprofesionales orientado al despliegue automatizado de topologías de red en EVE-NG mediante Python, consumo de API REST e Infraestructura como Código.*



*## Contenido*

*- `scripts/`: script principal de automatización*

*- `topologies/`: topologías en formato JSON*

*- `requirements.txt`: dependencias del proyecto*



*## Requisitos*

*- Python 3.x*

*- EVE-NG accesible por red*

*- Dependencias instaladas con:*



*```bash*

*pip install -r requirements.txt*
